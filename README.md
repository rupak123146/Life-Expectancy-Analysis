# Life Expectancy Analysis

## 📊 Project Description
This project analyzes and predicts life expectancy using machine learning techniques.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## 📁 Dataset
The dataset contains information about:
- GDP
- Population
- Schooling
- BMI
- Life Expectancy

## 📈 Features
- Data cleaning and preprocessing
- Data visualization (Histogram, Scatter Plot, Heatmap, Bar Chart)
- Prediction using Decision Tree Regression
- User input prediction

## 🧠 Model Used
Decision Tree Regressor is used to predict life expectancy based on input features.

## 📊 Output
- Graphs showing data distribution
- Predictions with Mean Absolute Error
- User-based prediction system

## 📌 Conclusion
Life expectancy is influenced by economic and health factors like GDP, education, and BMI.
